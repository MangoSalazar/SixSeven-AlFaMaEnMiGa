# SixSeven-AlFaMaEnMiGa

Sistema web escolar para la gestión de alumnos, docentes, materias, grupos y calificaciones.

## Estructura
- `docs/`: documentación del proyecto
- `db/`: scripts, diagramas y datos de prueba de base de datos
- `src/main/java/`: backend
- `src/main/resources/`: configuración
- `src/main/webapp/`: frontend
- `src/test/`: pruebas
